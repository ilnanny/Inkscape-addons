Inkscape Extension to generate a sheet cutting layout for a (truncated) cone.




This is an inkscape extension that unfolds a frustum (=truncated cone) or a cone (if cut diameter=0) and generates a sheet cutting layout or flat pattern projection

that can be rolled or bend up into a (truncated) cone shape.

Installation on Linux:

(sorry – on windows I have never tried…)

Copy the files „sheet_metal_conus.inx“ and „sheet_metal_conus.py“ to the directory „~/.config/inkscape/extensions“

or „<path_to_inkscape>/extensions“ and make the file „sheet_metal_conus.py“ executable with: „chmod a+x sheet_metal_conus.py“.


Using the extension:


Open inkscape and under the menu „Extensions->Sheet Metal→Sheet Metal Conus“ you will find following dialog:



  • Base diameter:

            the diameter of the cones base

  • Cut diameter:

            the diameter of the part of the cone that was cut away.

            If this parameter is zero a whole cone and not only a frustum (=cut cone) will be constructed.

  • Height:

            The height of the cone from the middle of the cones base to the middle of the cut away part or the top of the conus.

  • Units:

            Units can be either millimetre or inch. Note that also the line thickness will be in the same unit.

            Do not forget to adjust inkscape's default units in the documents properties dialog accordingly.

  • Line thickness:

            Thickness of the drawing line (in given unit).

  • Line colour:

            Colour of the drawing line. Can be given in numbers (e.g. '#000000') or by name (e.g. 'blue').

  • Verbose output:

            If checked a message box with all the calculated values will appear which can be useful for

            debugging purposes or if someone needs these values for drawing the layout by hand.
